### Edit 5: Add createDraftMutation and updateDraftMutation

  const createDraftMutation = useMutation({

- No `onError` callback here — error surfacing is handled at the call site (close handler) so the toast copy stays in one place.

### Edit 6: Add the unified close-save handler

Immediately after the two new mutations (and before the existing `createInvoiceMutation`), insert the save helper and close handler. Paste EXACTLY:

```typescript
  // D-01/D-03/D-04: single save path for both dialog close and (later) nav-away.
  // Skip entirely if no customer selected (D-04 — no empty drafts).
  const performDraftSave = async (): Promise<void> => {
    if (!selectedCustomer) return;

    const formData = getValues();
    const payload: DraftInvoicePayload = {
      type: formData.type,
      customerId: selectedCustomer.id,
      orderId: formData.orderId || undefined,
      issueDate: formData.issueDate || undefined,
      dueDate: formData.dueDate || undefined,
      notes: formData.notes,
      items: formData.items.map((item, idx) => ({
        type: item.type,
        description: item.description,
        quantity: item.quantity,
        unitPrice: item.unitPrice,
        unitCost: (item as any).unitCost,
        taxRate: item.taxRate,
        unit: item.unit,
        sortOrder: idx + 1,
      })),
    };

    try {
      if (draftIdRef.current) {
        await updateDraftMutation.mutateAsync({ id: draftIdRef.current, data: payload });
      } else {
        await createDraftMutation.mutateAsync(payload);
      }
    } catch {
      // UI-SPEC Copywriting Contract — exact strings are part of the contract.
      toast({
        variant: 'destructive',
        title: 'Entwurf konnte nicht gespeichert werden',
        description: 'Deine Eingaben sind noch sichtbar. Bitte versuche es erneut oder erstelle die Rechnung jetzt.',
      });
    }
  };

  // Single close entry point — unifies X button (via Dialog.Close → onOpenChange),
  // backdrop click (onOpenChange false), and the Abbrechen button (Pitfall 3).
  // Awaits the save so the dialog stays open long enough to complete the network call;
  // the save helper swallows errors and fires its own toast so this function never throws.
  const handleCloseRequested = async (): Promise<void> => {
    await performDraftSave();
    onClose();
  };
```
